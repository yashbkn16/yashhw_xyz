
# Numis (React · Light Theme) — by Yash Jain

Apple-clean, witty, minimal. Subtle 3D hovers, smooth scrolling, and an educational modal.

## Quickstart
```bash
npm i
npm run dev
```
Add your data:
1) Put images in `public/images/`
2) Fill `data/template.csv` OR edit `public/data/collection.json`
3) `npm run csv` to build JSON (from CSV)

## Deploy
Netlify/Cloudflare: build `vite build`, publish `dist/`.
GitHub Pages: push & point Pages to `dist/`.

## Personalize
Change accent in `src/styles.css` → `--accent`.
